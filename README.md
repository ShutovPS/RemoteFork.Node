# RemoteFork

**Программа является дополнением к приложению для тв** [*ForkPlayer*](http://forkplayer.tv/).

## Основные функции:
- Обработка запросов, недоступных в самом FP.
- DLNA сервер.
- Создание собственных плейлистов (пользовательских ссылок).
